## Como rodar a extensão localmente:
1. Digite “chrome://extensions” em uma janela do navegador Chrome para abrir a página de extensões.
2. No canto superior direito, ative “Modo de desenvolvedor” para habilitar a instalação de extensões descompactadas.
3. Por fim, clique em “Carregar extensão descompactada” e navegue até sua pasta de desenvolvimento; ou simplesmente arraste a pasta para a página para carregar a extensão.
